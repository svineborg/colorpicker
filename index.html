This is a tool for testing and generating 16 color palettes, like those used inxterm and other programs that might use the ANSI and aixterm-like colors. The color settings can be seen in the example below, and the .Xresources configuration below that.

[user@hostname ~]$ vim color_test.sh
1 #!/bin/sh
2 for C in {00.. 16}; do
3   echo -en "\033[38;5;${C} m $C "
4 done
5 echo -e "\e(B\e[m"
6 ~
[user@hostname ~]$ ./color_test.sh
